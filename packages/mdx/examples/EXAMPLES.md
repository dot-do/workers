# @hono/mdx Examples

This directory contains example applications demonstrating various uses of @hono/mdx middleware.

## Examples

### Blog Platform (`blog.ts`)
- Render blog posts with frontmatter metadata
- Extract title, author, date, tags
- Preview mode without HTML wrapper

### Documentation Site (`docs.ts`)
- File-based documentation routing
- Section organization
- Custom styling

## Running Examples

```bash
# Install dependencies
pnpm install

# Run example
pnpm tsx examples/blog.ts
```
