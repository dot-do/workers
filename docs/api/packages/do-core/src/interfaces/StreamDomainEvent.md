# Interface: StreamDomainEvent\<T\>

Defined in: [packages/do-core/src/event-store.ts:68](https://github.com/dot-do/workers/blob/c252c2d9415957e5ce9f5f849672bf290b40136f/packages/do-core/src/event-store.ts#L68)

Domain event structure for stream-based event sourcing.

Events are immutable records of things that happened in the system.
They are stored in streams (identified by streamId) with monotonic versions.

## Example

```typescript
const event: StreamDomainEvent<OrderPayload> = {
  id: 'evt-abc-123',
  streamId: 'order-456',
  type: 'OrderCreated',
  version: 1,
  timestamp: Date.now(),
  payload: { customerId: 'cust-789', total: 99.99 },
}
```
